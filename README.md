# CareerPay

CareerPay is an application for managing talent credit financing, employee stock ownership plans (ESOP), and payroll management.

## Tech Stack
- React
- Vite
- Tailwind CSS
- Lucide React
- Framer Motion
- Axios

## Features
- Company Dashboard
- Employee Management
- Payroll Processing
- ESOP Overview
- Talent Credit Financing
- Authentication (Sign In/Sign Up)
- Email Verification
